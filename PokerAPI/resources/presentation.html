Présentation

L'API Poker se compose d'un album de méthodes à appeler et de quelques points terminaux d'API.

Pour exécuter une action à l'aide de l'API Poker, vous devez sélectionner une convention d'appel, envoyer une requête à son point terminal en spécifiant un service des arguments, puis vous recevrez une réponse formatée.

Tous les formats de requête, énumérés à la page d'index API, utilisent une liste de paramètres nommés.

Le paramètre OBLIGATOIRE consumerKey permet de spécifier votre clé API.

Le paramètre OBLIGATOIRE signature permet de spécifier les arguments cryptés d'appel.

Le format de réponse respecte la norme JSON.

Les arguments, réponses et codes d'erreur de chaque méthode sont listés sur la page de données techniques de la méthode. Les méthodes correspondent à des listes sur la page d'index API.

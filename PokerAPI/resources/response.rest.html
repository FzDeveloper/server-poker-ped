Format de requête REST

REST est le format de requête le plus simple à utiliser – c'est aussi simple que les actions HTTP GET ou POST.

L'URL du point terminal REST est http://localhost:8080/PokerServer/rest/

Pour appeler des services "auth" (pour l'authentification), il faut procéder comm suit :

http://localhost:8080/PokerServer/rest/auth/{name_service}/{parameters}

Pour appeler des services "game", il faut procéder comm suit :

http://localhost:8080/PokerServer/rest/game/{name_service}/{parameters}

Pour appeler des services "player", il faut procéder comm suit :

http://localhost:8080/PokerServer/rest/player/{name_service}/{parameters}

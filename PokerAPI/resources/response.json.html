Format de réponse JSON

JSON, ou JavaScript Object Notation, est un format simple d'échange de données assimilable par l'ordinateur, qui facilite l'élaboration d'applications en Java avec l'API (il peut cependant être utilisé pour d'autres langages également !). Pour plus d'informations sur JSON, rendez-vous sur json.org

Représentation de l'objet

Voici un exemple de tags JSON :

{

 "foo": {
	"bar": "baz"
	}
}

Pour les éléments récurrents (tels que l'élément <player> lors de l'extraction d'une liste de joueurs), des crochets sont utilisés pour la valeur du membre de l'objet (la clé du membre représente le nom de l'élément). Chaque valeur comprise entre crochets est un objet représentant un élément enfant.

{
	"outer": {
	"players": [
	{
	"name": "Janet"
	},
	{
	"name": "John"
	}
	]
	}
}

Traitement des réponses

En Java, afficher la liste des joueurs d'un jeu est trés simple :

void jsonPokerApi(Response resp){

	if (rsp.stat != "ok"){

	// error -> do something
	return;
	}

	JSONObject json = toJSON(resp);
	List<String> players = json.get("players");

	for(String name : players) 
	DisplayList.add(name);
}

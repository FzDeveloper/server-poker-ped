poker.server.service.game.getGamesStatus

Retourne les informations qui concernent les tables de jeux en attente

REST Request

http://localhost:8080/PokerServer/rest/game/getGamesStatus/{consumerKey}

Authentification

Cette méthode n'exige pas une authentification.

Arguments

consumerKey : String (Obligatoire)
Your API application key. See here for more details.

Returns

gamesStatus: List<JSONObject>
La liste des tables de jeux en attente
Chaque JSONObject contient :
tableName : String
Name of  the game to start
gameTypeName : String
Nom du type du jeu de la table 
startGame: boolean
State of the game
playerNames: List<String>
The list of names of players' game
playerBudget: int
The number of the tokens that player has at the start of the game
buyIn: int
The money's player to play the game
smallBlind: int
The value of the small blind
bigBlind: int
The value of the big blind
prizePool: int
The prize of the game

Exemple de réponse 

{
 "stat":"ok",
 "tableName":"Labri_Texas_Holdem_1203203",
 "gameTypeName":"CashSitAndGo",
 "playerBudget":1500,
 "buyIn":10,
 "playerNames":["John","Laura","Markus","Claudio","Patxi"],
 "bigBlind":20
 "smallBlind":10
 "prizePool":50
}

Codes d'erreur

17: UNKNOWN_CONSUMER_KEY
Unknown consumer key

poker.server.service.game.connect

Connecter le joueur à la table choisie

REST Request

http://localhost:8080/PokerServer/rest/game/connect/{consumerKey}/{signature}

Authentification

Cette méthode exige une authentification.

Arguments

consumerKey: String (Obligatoire)
Your API application key. See here for more details.
signature: String (Obligatoire)
Tha signature generated from the String
 (secret+"&consumerKey&"+consumerKey+"&token&"+token+"&tableName&"+tableName+"&playerName&"+playerName)

Arguments pour la génération de la signature (à faire côté client)

secret: String (Obligatoire)
le code secret délivré au client lors de la génération du consumerKey
consumerKey: String (Obligatoire)
décrit plus haut
token: String (Obligatoire)
Le token d'accès qui a été envoyé à l'application
tableName: String (Obligatoire)
Le nom de la table à laquelle le joueur s'est connectée
playerName: String (Obligatoire)
Le nom du compte du joueur

Returns

tableName: String
Le nom de la table à laquelle le joueur s'est connectée

Exemple de réponse 

{
 "stat":"ok", 
 "tableName":"LabriTexasHoldem_55b09952-1ae3-4585-9f2f-d4ad69ff5154"}
}

Codes d'erreur

2: PLAYER_INGAME
The player is already affected in a game
3: GAME_NOT_READY_TO_START
The game is not ready to start
5: GAME_NOT_EXIST
The game doesn't exist
7: PLAYER_NOT_EXIST
The player doesn't exist
17: UNKNOWN_CONSUMER_KEY
Unknown consumer key
23: INVALID_SIGNATURE
Invalid signature
24: UNKNOWN_ACCESS_TOKEN
The access token is not valid
25: INCOMPATIBLE_ACCESS_TOKEN_CONSUMER
Incompatible access token and consumer key

poker.server.service.auth.getConsumerKey

Retourne une clé pour une application cliente qui servira d'identifiant 

REST Request

http://localhost:8080/PokerServer/rest/auth/getConsumerKey/{displayName}/

Authentification

Cette méthode n'exige pas une authentification.

Arguments

displayName: String (Obligatoire)
Le nom de l'application cliente

Returns

consumerKey: String
La clé qu'utilisera le client pour utiliser les services de l'API Poker
secret: String
Le secret (mot de passe) qui servira de crypter les requêtes vers l'API Poker

Exemple de réponse 

{
 "stat":"ok",
 "consumerKey":"4996a7d8-92d9-4f17-9853-fbb02ca950ac"
 "secret":"4ff6a7d8-00d9-4f17-fdfd53-vvb02cafdfd50ac"
}

Codes d'erreur

16: CONSUMEY_KEY_ERROR
Can not generate a consumer key

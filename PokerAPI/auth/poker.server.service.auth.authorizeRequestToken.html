Aide

poker.server.service.auth.authorizeRequestToken

Autoriser le request token après validation de l'utilisateur

REST Request

http://localhost:8080/PokerServer/rest/auth/authorizeRequestToken/{consumerKey}/{requestToken}

Authentification

Cette méthode exige une authentification.

Arguments

consumerKey: String (Obligatoire)
Votre clé API Poker
requestToken: String (Obligatoire)
Le request token validé par l'utilisateur

Returns

verifier: String
Un vérificateur qui servira à décrire si le request token a été validé (si différent de null)

Exemple de réponse 

{
 "stat":"ok"
 "verifier":"4ff6a7d8-00f9-4f17-fdfd53-vvb02cadf3440ac"
}

Codes d'erreur

17: UNKNOWN_CONSUMER_KEY
Unknown consumer key
18: UNKNOWN_REQUEST_TOKEN
Unknown request token
19: INCOMPATIBLE_RQTOKEN_CONSUMER
Incompatible request token and consumer key

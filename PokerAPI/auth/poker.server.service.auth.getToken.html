poker.server.service.auth.getToken

Retourne le token d'accès officiel aux service de l'API Poker

REST Request

http://localhost:8080/PokerServer/rest/auth/getToken/{consumerKey}/{requestToken}

Authentification

Cette méthode exige une authentification.

Arguments

consumerKey: String (Obligatoire)
Votre clé API Poker
requestToken: String (Obligatoire)
Le request token validé par l'utilisateur

Returns

oauthToken: String
Le request d'accès de l'utilisateur
oauthTokenSecret: String
Le secret du request d'accès de l'utilisateur

Exemple de réponse 

{
 "stat":"ok",
 "oauthToken":"90ecb0b7-52aa-4167-b073-e2ebbe68f376",
 "oauthTokenSecret":"77c89332-46f9-410a-af67-fc963d13bc26"
}

Codes d'erreur

17: UNKNOWN_CONSUMER_KEY
Unknown consumer key
18: UNKNOWN_REQUEST_TOKEN
Unknown request token
19: INCOMPATIBLE_RQTOKEN_CONSUMER
Incompatible request token and consumer key
20: REQ_TOKEN_NOT_VALID
The request token is not valid

poker.server.service.auth.getRequestToken

Retourne le request token qui doit être validé par un utilisateur pour l'application accéde à ses données

REST Request

http://localhost:8080/PokerServer/rest/auth/getRequestToken/{consumerKey}/{callBack}

Authentification

Cette méthode n'exige pas une authentification.

Arguments

consumerKey: String (Obligatoire)
Votre clé API Poker

Returns

oauthToken: String
Le request token de l'utilisateur
oauthTokenSecret: String
Le secret du request token de l'utilisateur

Exemple de réponse 

{
 "stat":"ok",
 "oauthToken":"90ecb0b7-52aa-4167-b073-e2ebbe68f376",
 "oauthTokenSecret":"77c89332-46f9-410a-af67-fc963d13bc26"
}

Codes d'erreur

17: UNKNOWN_CONSUMER_KEY
Unknown consumer key

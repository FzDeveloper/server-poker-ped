poker.server.service.gameType.addGameType

Ajouter un nouveau type de jeu au serveur de poker

REST Request

http://server:port/PokerServer/rest/gameType/addGameType/{consumerKey}/{gameName}/{playerNumber}/{playerTokens}/{buyIn}/{speakTime}/{smallBlind}
/{factorUpdateBlind}/{updateBlindTime}/{potType}/{numberOfWinners}

Authentification

Cette méthode n'exige pas une authentification.

Arguments

consumerKey: String (Obligatoire)
L'unique clé API Poker délivré à l'administrateur
gameName: String (Obligatoire)
Le nom du type de jeu
playerNumber: Integer (Obligatoire)
Le nombre de joueurs pour ce type de jeu
playerTokens: Integer (Obligatoire)
Le nombre de tokens que chaque joueur peut avoir au début du jeu
buyIn: Integer (Obligatoire)
Le montant fixé pour jouer à ce type de jeu
speakTime: Integer (Obligatoire)
Le temps qu'aura un joueur pour faire une action
smallBlind: Integer (Obligatoire)
Le montant de la small blind
factorUpdateBlind: Integer (Obligatoire)
Le facteur de multiplication à la mise à jour des blinds
updateBlindTime: Integer (Obligatoire)
L'unique clé API Poker délivré à l'administrateur
potType: Integer (Obligatoire)
Le type de pot à joueur (CASH ou TOKEN)
numberOfWinners: Integer (Obligatoire)
Le nombre des vainqueurs finaux 

Returns

stat: String
Le statut de la requête 

Exemple de réponse 

{
 "stat":"ok",
}

Codes d'erreur

17: UNKNOWN_CONSUMER_KEY
Unknown consumer key

poker.server.service.gameType.getAllGamesTypes

Retourne tous les types de jeu stockés dans la base de données

REST Request

http://localhost:8080/PokerServer/rest/gameType/getAllGamesTypes/{consumerKey}

Authentification

Cette méthode n'exige pas une authentification.

Arguments

consumerKey: String (Obligatoire)
L'unique clé API Poker délivré à l'administrateur

Returns

gamesStatus: JSONArray(JSONObject : gameType)
L'unique clé API Poker délivré à l'administrateur
Chaque gameType contient :
gameName: String 
Le nom du type de jeu
playerNumber: Integer 
Le nombre de joueurs pour ce type de jeu
playerTokens: Integer 
Le nombre de tokens que chaque joueur peut avoir au début du jeu
buyIn: Integer 
Le montant fixé pour jouer à ce type de jeu
speakTime: Integer 
Le temps qu'aura un joueur pour faire une action
smallBlind: Integer 
Le montant de la small blind
factorUpdateBlind: Integer 
Le facteur de multiplication à la mise à jour des blinds
updateBlindTime: Integer 
L'unique clé API Poker délivré à l'administrateur
potType: Integer 
Le type de pot à joueur (CASH ou TOKEN)
numberOfWinners: Integer 
Le nombre des vainqueurs finaux 

Exemple de réponse 

{
 "stat":"ok",
 "gamesTypes" : [
	"gameType" : {"gameName":"CashGame","playerNumber":8, "playerTokens":1500, "buyIn":10, "speakTime":30, "smallBlind":15, "factorUpdateBlind":2,
	"updateBlindTime":300, "potType":1, "numberOfWinners":4 
 ]
}

Codes d'erreur

17: UNKNOWN_CONSUMER_KEY
Unknown consumer key

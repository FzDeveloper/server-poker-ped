poker.server.service.gameType.deleteGameType

Supprimer le type de jeu de la base de données

REST Request

http://server:port/PokerServer/rest/gameType/deleteGameType/{consumerKey}/{gameName}

Authentification

Cette méthode n'exige pas une authentification.

Arguments

consumerKey: String (Obligatoire)
L'unique clé API Poker délivré à l'administrateur
gameName: String (Obligatoire)
Le nom du type de jeu

Returns

stat: String
Le statut de la requête 

Exemple de réponse 

{
 "stat":"ok",
}

Codes d'erreur

17: UNKNOWN_CONSUMER_KEY
Unknown consumer key
29: UNKNOWN_GAME_TYPE
Unknown game type
